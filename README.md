# VyomGarud - React Web App

##  Project Overview
VyomGarud is a React-based web application built as part of a frontend project.  
It includes responsive UI components and  Tailwind CSS integration

##  Tech Stack
- React.js (Vite)
- Tailwind CSS
- JavaScript (ES6+)
- Git & GitHub

##  Setup Instructions
1. Clone this repository  
   ```bash
   git clone https://github.com/Sai-Chandu-544/VyomGarud.git

2. Navigate to project folder
   cd VyomGarud
   
3. Install dependencies
   npm install

4. Run locally
   npm run dev
